WHAT IS CRAVE?

CRAVE is a Proof of Stake cryptocurrency based on Blackcoin PoS2.0. CRAVE includes technology including recent Bitcoin Core advancements such as BIP32 HD wallet addresses, dual key Stealth addresses, once-only transaction broadcasting, address index, and an advanced and friendly user interface.

TECHNOLOGY

CRAVE is the next generation cryptocurrency. While it's completely untraceable, it also provides its users safety from hackers.

SECP256K1

Integrate the ECDSA library by sipa to be implemented in Bitcoin core, significantly faster than the OpenSSL libs used currently.

ADRENALINE

Port masternodes from Darkcoin, modified to operate with dual key stealth addresses. Gain best of both worlds.

DARKPOOL / BLUR

Underground Colored Coins and Tor Hidden Service wallet. 30 day burn period is done.

ONCE ONLY TX

CRAVE wallet only broadcasts a transaction once. Default Bitcoin rebroadcasts tx’s wallet has economic interest in, which can be used to unmask and link IP’s and TX’s.
